# A2A Bridge Increment 3c — Claude adapter (warm Claude Code, kind-seam) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Claude as a registered agent via a new non-ACP `AgentBackend` (`bridge-claude` / `ClaudeCliBackend`) driving a warm interactive `claude --input-format stream-json --output-format stream-json` process per bridge `SessionId`, behind an `AgentKind` seam on the registry.

**Architecture:** `AgentEntry` gains a `kind` discriminator; the `bin/a2a-bridge` backend factory dispatches on it. `ClaudeCliBackend` mirrors `AcpBackend`'s warm-process machinery (per-session turn lock, lazy exactly-once `OnceCell` spawn, tolerant NDJSON reader→per-turn mpsc demux, `Supervised` process hygiene, bounded timeouts, `forget_session`-drops-only-stash) but is **one warm process per `SessionId`**, so it adds a bounded warm-pool (idle-TTL / `max_warm` LRU / hard `max_sessions` admission gate) plus a single `invalidate_slot` teardown primitive with a reaper-holds-turn-lock + post-lock-revalidation protocol that closes the reap⇄follow-up TOCTOU.

**Tech Stack:** Rust 2021 (edition pinned in workspace), `tokio`, `serde_json`, `async-trait`, `futures`, the existing `Supervised` supervisor (promoted to `bridge-core`), `bridge-core`/`bridge-registry`. `claude` CLI 2.1.159 (installed, logged in, no API key). No HTTP deps.

**Spec:** `docs/superpowers/specs/2026-05-31-a2a-bridge-v3c-claude-design.md` (rev3, twice dual-reviewed). Section refs below (`§3.1` etc.) point into it.

**Branch:** `feat/v3c-claude` off `main`.

**Naming/type contract (used consistently across all tasks):**
- `bridge_core::domain::AgentKind` — `enum { Acp (#[default]), ClaudeCli }`, `derive(Debug, Clone, Copy, PartialEq, Eq, Default)` (no serde — parsed from string in config like `Effort`).
- `bridge_core::error::BridgeError::AgentOverloaded` — new variant.
- `bridge_core::process::Supervised` — `Supervised` moved here from `bridge-acp` (shared by both process backends).
- New crate `bridge-claude` with modules: `config` (`ClaudeConfig`), `wire` (envelope + NDJSON parse), `proc` (`SessionProc`, `SessionSlot`, reader, `spawn_proc`), `backend` (`ClaudeCliBackend`).
- `ClaudeCliBackend::spawn(cmd: &str, config: ClaudeConfig) -> Result<Self, BridgeError>`.
- Extension keys: `idle_ttl_secs` (u64, default 3300), `max_warm` (usize, default 16), `max_sessions` (usize, default 64).

//! stream-json wire codec: serialize a user turn to stdin NDJSON; parse stdout
//! NDJSON into a small internal event enum (tolerant — unmodeled lines drop).
use serde_json::Value;

/// Parsed, modeled stdout event. Everything else (tool calls, usage, unknown
/// types) is dropped by `parse_line` returning `None`.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum ClaudeEvent {
    Init { session_id: String },
    Text(String),
    /// Terminal success result.
    ResultOk { stop_reason: Option<String> },
    /// Terminal error result (subtype e.g. error_max_turns / error_during_execution).
    ResultErr { subtype: String },
}

/// Serialize one user turn as the stdin NDJSON line (no trailing newline).
pub fn user_envelope(text: &str) -> String {
    serde_json::json!({
        "type": "user",
        "message": { "role": "user", "content": [ { "type": "text", "text": text } ] }
    })
    .to_string()
}

/// Parse one stdout line. Returns None for blank/unmodeled/unparseable lines.
pub fn parse_line(line: &str) -> Option<ClaudeEvent> {
    let line = line.trim();
    if line.is_empty() {
        return None;
    }
    let v: Value = serde_json::from_str(line).ok()?;
    match v.get("type")?.as_str()? {
        "system" if v.get("subtype").and_then(Value::as_str) == Some("init") => {
            let sid = v.get("session_id")?.as_str()?.to_string();
            Some(ClaudeEvent::Init { session_id: sid })
        }
        "assistant" => {
            // Concatenate all text blocks in message.content.
            let content = v.get("message")?.get("content")?.as_array()?;
            let mut text = String::new();
            for block in content {
                if block.get("type").and_then(Value::as_str) == Some("text") {
                    if let Some(t) = block.get("text").and_then(Value::as_str) {
                        text.push_str(t);
                    }
                }
            }
            if text.is_empty() {
                None
            } else {
                Some(ClaudeEvent::Text(text))
            }
        }
        "result" => {
            let subtype = v.get("subtype").and_then(Value::as_str).unwrap_or("success");
            if subtype == "success" {
                let stop = v
                    .get("stop_reason")
                    .and_then(Value::as_str)
                    .map(str::to_string);
                Some(ClaudeEvent::ResultOk { stop_reason: stop })
            } else {
                Some(ClaudeEvent::ResultErr {
                    subtype: subtype.to_string(),
                })
            }
        }
        _ => None,
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn envelope_shape() {
        assert_eq!(
            user_envelope("hi"),
            r#"{"message":{"content":[{"text":"hi","type":"text"}],"role":"user"},"type":"user"}"#
        );
    }

    #[test]
    fn parses_init() {
        let l = r#"{"type":"system","subtype":"init","session_id":"abc-123"}"#;
        assert_eq!(parse_line(l), Some(ClaudeEvent::Init { session_id: "abc-123".into() }));
    }

    #[test]
    fn parses_assistant_text() {
        let l = r#"{"type":"assistant","message":{"content":[{"type":"text","text":"PONG"}]}}"#;
        assert_eq!(parse_line(l), Some(ClaudeEvent::Text("PONG".into())));
    }

    #[test]
    fn parses_result_success_and_error() {
        assert_eq!(
            parse_line(r#"{"type":"result","subtype":"success"}"#),
            Some(ClaudeEvent::ResultOk { stop_reason: None })
        );
        assert_eq!(
            parse_line(r#"{"type":"result","subtype":"error_max_turns"}"#),
            Some(ClaudeEvent::ResultErr { subtype: "error_max_turns".into() })

    }

    #[test]

        assert_eq!(parse_line(r#"{"type":"usage","tokens":5}"#), None);

//! One warm `claude` process per bridge SessionId. Mirrors AcpBackend's
//! AgentSession (turn lock, lazy proc, tolerant reader → per-turn mpsc) but is
//! single-session-per-process, so it adds `terminated` (pool teardown flag) and
//! a per-turn timeout, and serializes turns with a single active mpsc sender.
use crate::config::ClaudeConfig;
use crate::wire::{self, ClaudeEvent};
use bridge_core::error::BridgeError;
use bridge_core::ports::Update;
use bridge_core::process::Supervised;
use std::sync::atomic::{AtomicBool, Ordering};
use std::sync::{Arc, Mutex as StdMutex};
use std::time::Instant;
use tokio::io::{AsyncBufReadExt, AsyncWriteExt, BufReader};
use tokio::sync::{mpsc, oneshot, Mutex, OnceCell};

/// Per-turn event routed from the reader to the active `prompt` stream.
#[derive(Debug)]
pub enum TurnEvent {
    Text(String),
    Done { stop_reason: String },
    Failed(BridgeError),
}

/// The warm process. One active turn at a time (serialized by `turn_lock`).
pub struct SessionProc {
    stdin: Mutex<tokio::process::ChildStdin>,
    pub turn_lock: Arc<Mutex<()>>,
    /// Set by the pool's `invalidate_slot` (reaper/LRU/cancel/timeout). A `prompt`
    /// that acquired the turn lock AFTER a reap observes this and respawns (§3.2).
    pub terminated: AtomicBool,
    /// Cancel latch: when set, an EOF / error `result` for the in-flight turn maps
    /// to Canceled, not Failed (§4 cancel precedence).
    pub cancel_requested: AtomicBool,
    /// The single active turn's sender (one turn at a time).
    turn_tx: StdMutex<Option<mpsc::UnboundedSender<TurnEvent>>>,
    supervised: StdMutex<Option<Supervised>>,
    pub claude_session_id: StdMutex<Option<String>>,
    pub last_used: StdMutex<Instant>,
}

impl SessionProc {
    pub fn touch(&self) {
        if let Ok(mut t) = self.last_used.lock() {
            *t = Instant::now();
        }
    }
    pub fn idle_for(&self) -> std::time::Duration {
        self.last_used
            .lock()
            .map(|t| t.elapsed())
            .unwrap_or_default()
    }
    /// Write the user envelope for a turn (caller holds the turn lock).
    pub async fn write_turn(&self, text: &str) -> Result<(), BridgeError> {
        let line = wire::user_envelope(text);
        let mut w = self.stdin.lock().await;
        w.write_all(line.as_bytes())
            .await
            .map_err(|_| BridgeError::AgentCrashed)?;
        w.write_all(b"\n").await.map_err(|_| BridgeError::AgentCrashed)?;
        w.flush().await.map_err(|_| BridgeError::AgentCrashed)?;
        Ok(())
    }
    /// Register the active turn's sender; returns the receiver the stream drains.
    pub fn begin_turn(&self) -> mpsc::UnboundedReceiver<TurnEvent> {
        let (tx, rx) = mpsc::unbounded_channel();
        if let Ok(mut g) = self.turn_tx.lock() {
            *g = Some(tx);
        }
        rx
    }
    pub fn end_turn(&self) {
        if let Ok(mut g) = self.turn_tx.lock() {
            *g = None;
        }
    }
    /// Terminate the child (idempotent take-once). Sets `terminated` first so a
    /// racing `prompt` revalidation observes it.
    pub async fn terminate(&self, grace: std::time::Duration) {
        self.terminated.store(true, Ordering::SeqCst);
        let sup = self.supervised.lock().ok().and_then(|mut g| g.take());
        if let Some(sup) = sup {
            sup.terminate(grace).await;
        }
    }
    fn route(&self, ev: TurnEvent) {
        if let Ok(g) = self.turn_tx.lock() {
            if let Some(tx) = g.as_ref() {
                let _ = tx.send(ev);
            }
        }
    }
}

/// A map slot holding the lazily-minted proc. `OnceCell` cannot be reset, so the
/// pool teardown REMOVES the slot from the map (a fresh prompt re-inserts a new
/// slot → cold respawn).
pub struct SessionSlot {
    pub proc: OnceCell<Arc<SessionProc>>,
}

impl SessionSlot {
    pub fn new() -> Self {
        Self { proc: OnceCell::new() }
    }
}

/// Spawn one warm `claude` process and its reader task; await the init line
/// (bounded by `init_timeout`) so we capture the session id and surface an
/// auth/trust failure as a bounded error rather than a hang (§4).
pub async fn spawn_proc(cmd: &str, cfg: &ClaudeConfig) -> Result<Arc<SessionProc>, BridgeError> {
    let mut args: Vec<String> = vec![
        "--input-format".into(),
        "stream-json".into(),
        "--output-format".into(),
        "stream-json".into(),
        "--verbose".into(),
    ];
    if let Some(m) = &cfg.model {
        args.push("--model".into());
        args.push(m.clone());
    }
    args.extend(cfg.extra_args.iter().cloned());
    let arg_refs: Vec<&str> = args.iter().map(String::as_str).collect();

    // Supervised spawns with process_group(0) + kill_on_drop + piped stdio.
    let mut sup = Supervised::spawn(cmd, &arg_refs).map_err(|_| BridgeError::AgentCrashed)?;
    let child = sup.child_mut();
    let stdin = child.stdin.take().ok_or(BridgeError::AgentCrashed)?;
    let stdout = child.stdout.take().ok_or(BridgeError::AgentCrashed)?;

    let (init_tx, init_rx) = oneshot::channel::<String>();
    let proc = Arc::new(SessionProc {
        stdin: Mutex::new(stdin),
        turn_lock: Arc::new(Mutex::new(())),
        terminated: AtomicBool::new(false),
        cancel_requested: AtomicBool::new(false),
        turn_tx: StdMutex::new(None),
        supervised: StdMutex::new(Some(sup)),
        claude_session_id: StdMutex::new(None),
        last_used: StdMutex::new(Instant::now()),
    });

    // Reader task: NDJSON line loop, tolerant drop, per-turn routing.
    let reader_proc = Arc::clone(&proc);
    tokio::spawn(async move {
        let mut lines = BufReader::new(stdout).lines();
        let mut init_tx = Some(init_tx);
        loop {
            match lines.next_line().await {
                Ok(Some(line)) => {
                    if let Some(ev) = wire::parse_line(&line) {
                        match ev {
                            ClaudeEvent::Init { session_id } => {
                                if let Ok(mut g) = reader_proc.claude_session_id.lock() {
                                    *g = Some(session_id.clone());
                                }
                                if let Some(tx) = init_tx.take() {
                                    let _ = tx.send(session_id);
                                }
                            }
                            ClaudeEvent::Text(t) => reader_proc.route(TurnEvent::Text(t)),
                            ClaudeEvent::ResultOk { stop_reason } => reader_proc.route(
                                TurnEvent::Done {
                                    stop_reason: stop_reason.unwrap_or_else(|| "end_turn".into()),
                                },
                            ),
                            ClaudeEvent::ResultErr { subtype } => {
                                if reader_proc.cancel_requested.load(Ordering::SeqCst) {
                                    reader_proc.route(TurnEvent::Done {
                                        stop_reason: "cancelled".into(),
                                    });
                                } else {
                                    tracing::warn!(subtype, "claude result error");
                                    reader_proc.route(TurnEvent::Failed(BridgeError::AgentCrashed));
                                }
                            }
                        }
                    }
                }
                Ok(None) | Err(_) => {
                    // EOF / read error. If a cancel is pending, the in-flight turn
                    // is Canceled; otherwise it's a crash mid-turn.
                    reader_proc.terminated.store(true, Ordering::SeqCst);
                    if reader_proc.cancel_requested.load(Ordering::SeqCst) {
                        reader_proc.route(TurnEvent::Done { stop_reason: "cancelled".into() });
                    } else {
                        reader_proc.route(TurnEvent::Failed(BridgeError::AgentCrashed));
                    }
                    break;
                }
            }
        }
    });

    // Bounded wait for the init line.
    match tokio::time::timeout(cfg.init_timeout, init_rx).await {
        Ok(Ok(_sid)) => Ok(proc),
        // Channel dropped (reader hit EOF before init) → not authenticated / trust prompt / crash.
        Ok(Err(_)) => {
            proc.terminate(cfg.cancel_grace).await;
            Err(BridgeError::AgentNotAuthenticated)
        }
        Err(_) => {
            proc.terminate(cfg.cancel_grace).await;
            Err(BridgeError::AgentNotAuthenticated)
        }
    }
}

//! ClaudeCliBackend: warm `claude` process per SessionId behind the AgentBackend
//! trait, with a bounded warm-pool (idle-TTL / max_warm LRU / hard max_sessions)
//! and a single `invalidate_slot` teardown reused by reap/LRU/cancel/timeout.
use crate::config::ClaudeConfig;
use crate::proc::{spawn_proc, SessionProc, SessionSlot, TurnEvent};
use async_trait::async_trait;
use bridge_core::domain::{EffectiveConfig, Part};
use bridge_core::error::BridgeError;
use bridge_core::ids::SessionId;
use bridge_core::ports::{AgentBackend, BackendStream, Update};
use std::collections::HashMap;
use std::sync::atomic::Ordering;
use std::sync::{Arc, Mutex as StdMutex};
use tokio::sync::Mutex;

pub(crate) struct Inner {
    pub cmd: String,
    pub config: ClaudeConfig,
    pub sessions: Mutex<HashMap<SessionId, Arc<SessionSlot>>>,
    pub session_cfg: StdMutex<HashMap<SessionId, EffectiveConfig>>,
}

pub struct ClaudeCliBackend {
    pub(crate) inner: Arc<Inner>,
    reaper: StdMutex<Option<tokio::task::JoinHandle<()>>>,
}

impl ClaudeCliBackend {
    /// Build the backend and start the warm-pool reaper.
    pub async fn spawn(cmd: &str, config: ClaudeConfig) -> Result<Self, BridgeError> {
        let inner = Arc::new(Inner {
            cmd: cmd.to_string(),
            config,
            sessions: Mutex::new(HashMap::new()),
            session_cfg: StdMutex::new(HashMap::new()),
        });
        let reaper = crate::backend::reaper::spawn_reaper(Arc::clone(&inner));
        Ok(Self {
            inner,
            reaper: StdMutex::new(Some(reaper)),
        })
    }

    /// Count live (non-terminated, minted) procs — the hard admission gate basis.
    pub(crate) async fn live_count(inner: &Inner) -> usize {
        let map = inner.sessions.lock().await;
        map.values()
            .filter(|s| {
                s.proc
                    .get()
                    .map(|p| !p.terminated.load(Ordering::SeqCst))
                    .unwrap_or(false)
            })
            .count()
    }

    /// Remove a slot from the map by key (the OnceCell reset) and terminate its
    /// proc. THE single teardown primitive (reap/LRU/cancel/turn-timeout). The
    /// caller is responsible for the turn-lock discipline (reaper holds it; cancel
    /// sets the latch first).
    pub(crate) async fn invalidate_slot(inner: &Inner, session: &SessionId) {
        let slot = {
            let mut map = inner.sessions.lock().await;
            map.remove(session)
        };
        if let Some(slot) = slot {
            if let Some(proc) = slot.proc.get() {
                proc.terminate(inner.config.cancel_grace).await;
            }
        }
    }

    /// Get-or-insert the slot for a session. On a NEW slot (would spawn), enforce
    /// the hard `max_sessions` admission gate first.
    async fn slot_for(&self, session: &SessionId) -> Result<Arc<SessionSlot>, BridgeError> {
        {
            let map = self.inner.sessions.lock().await;
            if let Some(s) = map.get(session) {
                return Ok(Arc::clone(s));
            }
        }
        // New session → admission gate (§3.3).
        if Self::live_count(&self.inner).await >= self.inner.config.max_sessions {
            // Try to reap one idle proc to make room.
            if !reaper::reap_one_idle(&self.inner).await {
                return Err(BridgeError::AgentOverloaded);
            }
        }
        let mut map = self.inner.sessions.lock().await;
        let slot = map
            .entry(session.clone())
            .or_insert_with(|| Arc::new(SessionSlot::new()));
        Ok(Arc::clone(slot))
    }

    /// Get the slot's warm proc, lazily spawning exactly once.
    async fn proc_for(
        &self,
        slot: &Arc<SessionSlot>,
        eff: Option<&EffectiveConfig>,
    ) -> Result<Arc<SessionProc>, BridgeError> {
        let mut cfg = self.inner.config.clone();
        if let Some(e) = eff {
            if let Some(m) = &e.model {
                cfg.model = Some(m.clone());
            }
        }
        let cmd = self.inner.cmd.clone();
        slot.proc
            .get_or_try_init(|| async { spawn_proc(&cmd, &cfg).await })
            .await
            .cloned()
    }
}

#[async_trait]
impl AgentBackend for ClaudeCliBackend {
    async fn prompt(
        &self,
        session: &SessionId,
        parts: Vec<Part>,
    ) -> Result<BackendStream, BridgeError> {
        let text = parts_to_text(&parts);
        let eff = self
            .inner
            .session_cfg
            .lock()
            .ok()
            .and_then(|m| m.get(session).cloned());

        // Resolve slot + proc, then the post-lock revalidation loop (§3.2): if the
        // proc we got was terminated by a reap in the window before we held the
        // turn lock, drop it, invalidate the slot, and respawn — at most twice.
        let mut attempts = 0;
        let (proc, turn_guard) = loop {
            attempts += 1;
            let slot = self.slot_for(session).await?;
            let proc = self.proc_for(&slot, eff.as_ref()).await?;
            let guard = Arc::clone(&proc.turn_lock).lock_owned().await;
            if !proc.terminated.load(Ordering::SeqCst) {
                break (proc, guard);
            }
            drop(guard);
            Self::invalidate_slot(&self.inner, session).await;
            if attempts >= 3 {
                return Err(BridgeError::AgentCrashed);
            }
            // loop → fresh slot + cold respawn
        };

        // Drive the turn.
        let rx = proc.begin_turn();
        proc.cancel_requested.store(false, Ordering::SeqCst);
        proc.write_turn(&text).await?;
        proc.touch();

        let turn_timeout = self.inner.config.turn_timeout;
        let proc_for_stream = Arc::clone(&proc);
        let stream = async_stream::stream! {
            // Move the turn guard + a deadline into the stream so the lock is held
            // for the whole turn and released when the stream ends (mirrors AcpBackend).
            let _guard = turn_guard;
            let mut rx = rx;
            let deadline = tokio::time::sleep(turn_timeout);
            tokio::pin!(deadline);
            loop {
                tokio::select! {
                    maybe = rx.recv() => match maybe {
                        Some(TurnEvent::Text(t)) => yield Ok(Update::Text(t)),
                        Some(TurnEvent::Done { stop_reason }) => {
                            proc_for_stream.touch();
                            proc_for_stream.end_turn();
                            yield Ok(Update::Done { stop_reason });
                            break;
                        }
                        Some(TurnEvent::Failed(e)) => {
                            proc_for_stream.end_turn();
                            yield Err(e);
                            break;
                        }
                        None => {
                            proc_for_stream.end_turn();
                            yield Err(BridgeError::AgentCrashed);
                            break;
                        }
                    },
                    _ = &mut deadline => {
                        // Per-turn timeout: tear the proc down so a hung turn can't
                        // hold the lock forever; surface Failed.
                        proc_for_stream.end_turn();
                        proc_for_stream.terminate(turn_timeout).await;
                        yield Err(BridgeError::AgentCrashed);
                        break;
                    }
                }
            }
        };
        Ok(Box::pin(stream))
    }

    async fn cancel(&self, session: &SessionId) -> Result<(), BridgeError> {
        // Filled in Task 10.
        let _ = session;
        Ok(())
    }

    async fn configure_session(
        &self,
        session: &SessionId,
        cfg: &EffectiveConfig,
    ) -> Result<(), BridgeError> {
        if let Ok(mut m) = self.inner.session_cfg.lock() {
            m.insert(session.clone(), cfg.clone());
        }
        Ok(())
    }

    /// DROPS ONLY THE CONFIG STASH — never kills the process (mirrors AcpBackend
    /// acp_backend.rs:1370). The warm proc survives across same-TaskId turns; only
    /// the pool reaps it. THIS is the blocker fix (§3.1).
    async fn forget_session(&self, session: &SessionId) {
        if let Ok(mut m) = self.inner.session_cfg.lock() {
            m.remove(session);
        }
    }

    async fn retire(&self) -> Result<(), BridgeError> {
        // Filled in Task 11.
        Ok(())
    }
}

    // The backend only sends text envelopes; concatenate the parts' text.
    parts.iter().map(|p| p.text.as_str()).collect::<Vec<_>>().join("")
}

```rust
//! Warm-pool reaper: idle-TTL + max_warm LRU eviction. Every termination path
//! goes through the reaper protocol — `try_lock` the proc's turn lock and HOLD it
//! while invalidating (so a racing prompt observes `terminated` after it finally
//! gets the lock). A proc mid-turn (try_lock fails) is skipped this pass.
use super::{ClaudeCliBackend, Inner};
use crate::proc::SessionProc;
use bridge_core::ids::SessionId;
use std::sync::atomic::Ordering;
use std::sync::Arc;
use std::time::Duration;

/// One candidate: its session id + proc + idle duration.
fn snapshot_live(map: &std::collections::HashMap<SessionId, Arc<crate::proc::SessionSlot>>)
    -> Vec<(SessionId, Arc<SessionProc>, Duration)>
{
    map.iter()
        .filter_map(|(sid, slot)| {
            slot.proc.get().and_then(|p| {
                if p.terminated.load(Ordering::SeqCst) {
                    None
                } else {
                    Some((sid.clone(), Arc::clone(p), p.idle_for()))
                }
            })
        })
        .collect()
}

/// Attempt to invalidate `session`'s proc under the reaper protocol. Returns true
/// if it was reaped (turn lock was free), false if it was mid-turn (skipped).
async fn try_reap(inner: &Inner, session: &SessionId, proc: &Arc<SessionProc>) -> bool {
    // try_lock the turn lock: success ⇒ no turn in flight ⇒ safe to invalidate.
    let guard = match proc.turn_lock.try_lock() {
        Ok(g) => g,
        Err(_) => return false, // mid-turn → skip
    };
    // Hold the lock across invalidation so a prompt that cloned the proc Arc will,
    // on finally acquiring the lock, observe `terminated` and respawn.
    proc.terminated.store(true, Ordering::SeqCst);
    ClaudeCliBackend::invalidate_slot(inner, session).await;
    drop(guard);
    true
}

/// Reap exactly one idle proc (used by the admission gate to make room). Picks the
/// most-idle reapable proc.
pub(crate) async fn reap_one_idle(inner: &Inner) -> bool {
    let mut cands = {
        let map = inner.sessions.lock().await;
        snapshot_live(&map)
    };
    cands.sort_by_key(|(_, _, idle)| std::cmp::Reverse(*idle));
    for (sid, proc, _) in cands {
        if try_reap(inner, &sid, &proc).await {
            return true;
        }
    }
    false
}

async fn reap_pass(inner: &Inner) {
    let cands = {
        let map = inner.sessions.lock().await;
        snapshot_live(&map)
    };
    // 1) Idle-TTL: reap anything idle past the TTL.
    let ttl = inner.config.idle_ttl;
    for (sid, proc, idle) in &cands {
        if *idle > ttl {
            let _ = try_reap(inner, sid, proc).await;
        }
    }
    // 2) max_warm LRU: if still over the idle-retention cap, evict most-idle first.
    let mut live = {
        let map = inner.sessions.lock().await;
        snapshot_live(&map)
    };
    if live.len() > inner.config.max_warm {
        live.sort_by_key(|(_, _, idle)| std::cmp::Reverse(*idle));
        let over = live.len() - inner.config.max_warm;
        let mut evicted = 0;
        for (sid, proc, _) in live {
            if evicted >= over {
                break;
            }
            if try_reap(inner, &sid, &proc).await {
                evicted += 1;
            }
        }
    }
}

pub(crate) fn spawn_reaper(inner: Arc<Inner>) -> tokio::task::JoinHandle<()> {
    let interval = inner.config.reaper_interval;
    tokio::spawn(async move {
        let mut tick = tokio::time::interval(interval);
        tick.set_missed_tick_behavior(tokio::time::MissedTickBehavior::Skip);
        loop {
            tick.tick().await;
            reap_pass(&inner).await;
        }
    })
}

/// Test-only: run a single reap pass synchronously.

pub async fn reap_pass_for_test(inner: &Inner) {
    reap_pass(inner).await

    let gate = Arc::new(Notify::new());
    be.set_race_gate(gate.clone());
    let be2 = be.clone();

        let mut st = be2.prompt(&s2, tp("What number?")).await.unwrap();
        let mut out = Vec::new();
        while let Some(i) = st.next().await {
            match i { Ok(Update::Text(t)) => out.push(t), Ok(Update::Done{..}) => break, Err(e) => panic!("spurious crash: {e}"), _ => {} }
        }
        out
    });
    // While the prompt is parked in the TOCTOU window, reap its proc.
    tokio::time::sleep(std::time::Duration::from_millis(50)).await;
    be.reap_now_force(&s).await; // reap THIS session unconditionally (test hook)
    // Release the parked prompt: it must observe terminated → respawn → succeed.
    gate.notify_one();
    let out = prompt_task.await.unwrap();
    // After a cold respawn the fake has no memory of "7", so it answers "OK" — the
    // point is the turn SUCCEEDS (no spurious crash), proving revalidation+respawn.
    assert!(!out.is_empty(), "respawned turn produced output (no spurious crash)");
}
```
