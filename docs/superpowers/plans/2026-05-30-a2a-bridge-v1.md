# A2A Bridge v1 (Walking Skeleton) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single Rust binary that exposes the Kiro CLI as an A2A-compliant network service end-to-end (inbound), with all extraction seams (including a stubbed outbound `DelegationPort`) defined and invoked.

**Architecture:** Hexagonal/ports-and-adapters. A domain-only `bridge-core` (Task/Session typestate, translator, ports) is driven by adapters: `bridge-acp` (Kiro subprocess over ACP NDJSON), `bridge-a2a-inbound` (A2A server via `a2aproject/a2a-rs`), `bridge-a2a-outbound` (stub), `bridge-store` (SQLite), `bridge-policy` (auth/permission, no routing), `bridge-observ` (tracing). Parse-don't-validate at both wire edges; typestate makes illegal protocol states uncompilable.

**Tech Stack:** Rust 2021, `tokio`, `agent-client-protocol` (ACP), `a2aproject/a2a-rs` (A2A v1, generated ProtoJSON), `sqlx`/`rusqlite`, `tracing`, `serde`/`serde_json`, `thiserror`, `trybuild` (compile-fail tests), `cargo-llvm-cov` (coverage), `cargo-deny` (license/advisory).

**Spec:** `docs/superpowers/specs/2026-05-29-a2a-bridge-v1-design.md`. Inline tags `[Codex N]`/`[Claude X]` trace to accepted review findings.

## Coverage Policy (enforced from Task 0)

Foundation correctness is gated by coverage, per the user directive:

- **`bridge-core` (domain, typestate, translator) and `bridge-acp` framing: ≥ 90% line coverage.** These hold the parse boundary and the anti-corruption logic — the bug-dense surface (v3 §5.4).
- **Workspace overall: ≥ 85% line coverage.**
- CI fails the build if either threshold regresses. Tool: `cargo llvm-cov --workspace --fail-under-lines 85` plus a per-crate check for the two foundation crates.
- Coverage is a floor, not a target — every Task below is TDD (test first), so coverage is a side effect of discipline, and the gate catches accidental gaps.

macro_rules! id_newtype {

// error.rs
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum A2aState { Submitted, Working, InputRequired, AuthRequired, Completed, Failed, Canceled }

#[derive(Debug, thiserror::Error, Clone, PartialEq, Eq)]
pub enum BridgeError {
    #[error("a2a version mismatch")] A2aVersionMismatch,
    #[error("invalid request: {field}")] InvalidRequest { field: &'static str },
    #[error("task not found")] TaskNotFound,
    #[error("session not found")] SessionNotFound,
    #[error("auth required")] AuthRequired { request_id: String },
    #[error("permission required")] PermissionRequired { request_id: String },
    #[error("permission denied")] PermissionDenied,
    #[error("agent not authenticated")] AgentNotAuthenticated,
    #[error("model not available")] ModelNotAvailable,
    #[error("cancel timeout")] CancelTimeout,
    #[error("frame error")] FrameError,
    #[error("message too large")] MessageTooLarge,
    #[error("agent crashed")] AgentCrashed,
    #[error("upstream a2a error")] UpstreamA2aError,
    #[error("store failure")] StoreFailure,
    #[error("invalid state transition")] InvalidStateTransition,
}
impl BridgeError {
    pub fn a2a_state(&self) -> A2aState {
        use BridgeError::*; use A2aState as S;
        match self {
            AuthRequired{..} | AgentNotAuthenticated => S::AuthRequired,
            PermissionRequired{..} => S::InputRequired,
            CancelTimeout => S::Canceled,
            A2aVersionMismatch | InvalidRequest{..} | TaskNotFound | SessionNotFound => S::Failed,
            _ => S::Failed,
        }
    }
    pub fn is_resumable(&self) -> bool {
        matches!(self, BridgeError::AuthRequired{..} | BridgeError::PermissionRequired{..})
    }
}
```

Add `pub mod error;` to `lib.rs`.

// task.rs
use crate::ids::TaskId;
pub enum TaskState { Submitted, Working, InputRequired, AuthRequired, Completed, Failed, Canceled }

pub struct Task<S> { id: TaskId, _s: std::marker::PhantomData<S>, runtime_state: TaskState }
pub struct Submitted; pub struct Working; pub struct InputRequired; pub struct AuthRequired;
pub struct Terminal;

impl Task<Submitted> {
    pub fn submitted(id: TaskId) -> Self { Self{ id, _s: Default::default(), runtime_state: TaskState::Submitted } }
    pub fn start(self) -> Task<Working> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::Working } }
}
impl Task<Working> {
    pub fn suspend_input(self, _request_id: String) -> Task<InputRequired> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::InputRequired } }
    pub fn suspend_auth(self, _request_id: String) -> Task<AuthRequired> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::AuthRequired } }
    pub fn complete(self) -> Task<Terminal> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::Completed } }
    pub fn fail(self) -> Task<Terminal> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::Failed } }
    pub fn cancel(self) -> Task<Terminal> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::Canceled } }
}
impl Task<InputRequired> { pub fn resume(self) -> Task<Working> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::Working } } }
impl Task<AuthRequired>  { pub fn resume(self) -> Task<Working> { Task{ id:self.id, _s:Default::default(), runtime_state: TaskState::Working } } }
impl<S> Task<S> { pub fn state(&self) -> &TaskState { &self.runtime_state } pub fn id(&self) -> &TaskId { &self.id } }
```

```rust
// framing.rs
use tokio::io::{AsyncRead, AsyncReadExt};
use bridge_core::error::BridgeError;

pub struct FrameReader<R> { inner: R, buf: Vec<u8>, max: usize }
impl<R: AsyncRead + Unpin> FrameReader<R> {
    pub fn new(inner: R, max: usize) -> Self { Self { inner, buf: Vec::new(), max } }
    pub async fn next(&mut self) -> Option<Result<serde_json::Value, BridgeError>> {
        loop {
            if let Some(pos) = self.buf.iter().position(|&b| b == b'\n') {
                let line: Vec<u8> = self.buf.drain(..=pos).collect();
                let line = &line[..line.len()-1];
                if line.is_empty() { continue; }
                if line.len() > self.max { return Some(Err(BridgeError::MessageTooLarge)); }
                return Some(serde_json::from_slice(line).map_err(|_| BridgeError::FrameError));
            }
            if self.buf.len() > self.max { return Some(Err(BridgeError::MessageTooLarge)); }
            let mut tmp = [0u8; 4096];
            match self.inner.read(&mut tmp).await {
                Ok(0) => return None,
                Ok(n) => self.buf.extend_from_slice(&tmp[..n]),
                Err(_) => return Some(Err(BridgeError::FrameError)),
            }
        }
    }
}
```

pub struct Supervised { child: Child, pid: u32 }
impl Supervised {
    pub fn spawn(prog: &str, args: &[&str]) -> std::io::Result<Self> {
        let mut child = Command::new(prog).args(args)
            .stdin(Stdio::piped()).stdout(Stdio::piped()).stderr(Stdio::piped())
            .process_group(0).kill_on_drop(true).spawn()?;
        let pid = child.id().expect("pid");
        Ok(Self { child, pid })
    }
    pub fn pid(&self) -> u32 { self.pid }
    pub async fn terminate(mut self, grace: std::time::Duration) {
        // SIGTERM to the group, wait grace, SIGKILL fallback, then reap
        unsafe { libc::kill(-(self.pid as i32), libc::SIGTERM); }
        let _ = tokio::time::timeout(grace, self.child.wait()).await;
        let _ = self.child.start_kill();
        let _ = self.child.wait().await; // reap
    }
}
