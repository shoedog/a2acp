# docs/history

Frozen one-shot dev-process artifacts (slice/increment/livegate/arch-review configs
and prompts used to build this repo). They are machine-specific and reference
paths, dates, and reviewers that no longer apply — never rerun them. Git history
(and the still-open `docs/superpowers/` specs/handoffs) is the authoritative
record of what they did; these files are kept only for archival grep-ability.
