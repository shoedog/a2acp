**(g) Workflow/Prompt Management Ruling**

Ruling: use a hybrid.

- Per-target-repo workflows and prompts should live in the using repo, normally `tools/a2a-bridge/configs/` and `tools/a2a-bridge/prompts/`. Evidence: `init` embeds curated generic prompts [main.rs](/Users/wesleyjinks/code/a2a-bridge/bin/a2a-bridge/src/main.rs:3874) and emits named prompt registry workflows [main.rs](/Users/wesleyjinks/code/a2a-bridge/bin/a2a-bridge/src/main.rs:3968). Confidence 90.
